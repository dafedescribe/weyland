## Design Style Guide

### Color Palette
- **Primary**: `#10b981` (Emerald Green - WhatsApp theme)
- **Secondary**: `#059669` (Darker green)
- **Accent**: `#fbbf24` (Amber - for highlights)
- **Dark**: `#0f172a` (Slate 900)
- **Light**: `#f8fafc` (Slate 50)
- **Text Primary**: `#1e293b` (Slate 800)
- **Text Secondary**: `#64748b` (Slate 500)

### Typography
- **Headings**: Inter or Poppins (Bold, 700-800 weight)
- **Body**: Inter or System UI (Regular, 400-500 weight)
- **Code/Stats**: JetBrains Mono or Fira Code

### Design Principles
1. **Modern & Clean**: Lots of whitespace, clear hierarchy
2. **Mobile-First**: Perfect on phones, great on desktop
3. **Micro-Animations**: Subtle hover effects, smooth transitions
4. **Glassmorphism**: For cards and sections (backdrop-blur)
5. **Bold Typography**: Large, impactful headings
6. **Vibrant Gradients**: Green to emerald, amber accents
7. **Dark Mode Support**: Toggle between light/dark themes

Every CTA button should open WhatsApp with pre-filled message:
```javascript
const openWhatsApp = (message = "Hi! I want to start creating links") => {
  const url = `https://wa.me/${siteConfig.whatsappNumber}?text=${encodeURIComponent(message)}`
  window.open(url, '_blank')
}
```

### 2. Video Modals
Clicking video thumbnails opens modal with embedded player:
- YouTube/Vimeo embed
- Close button (X)

- Full-size image
- Zoom in/out
- Caption
- Navigation arrows
- Close button

### 4. Smooth Scrolling
All navigation links use smooth scroll behavior

### 5. Cost Calculator
Interactive slider that calculates daily/weekly/monthly costs based on number of links

3. **Hover Effects**: 
   - Cards: Lift + shadow increase
   - Buttons: Scale + color shift
   - Links: Underline animation
4. **CTA Buttons**: Pulse animation loop
5. **Stats Counter**: Count-up animation when in view
6. **Video Thumbnails**: Play icon scale on hover

## Responsive Breakpoints

- **Mobile**: < 640px (1 column layouts)
- **Tablet**: 640px - 1024px (2 column layouts)
- **Desktop**: > 1024px (3-4 column layouts)
- **Large Desktop**: > 1536px (Max width container)

## Performance Optimizations

1. **Lazy Loading**: Images and videos load on scroll
2. **Image Optimization**: WebP format, multiple sizes
3. **Code Splitting**: Load sections as needed
4. **Minification**: CSS and JS bundled and minified
5. **Caching**: Service worker for static assets
6. **CDN**: Host media on CDN (future)

---

## Accessibility

1. **Semantic HTML**: Proper heading hierarchy
2. **Alt Text**: All images have descriptive alt text
3. **Keyboard Navigation**: Tab through all interactive elements
4. **ARIA Labels**: For icons and dynamic content
5. **Color Contrast**: WCAG AA compliance
6. **Focus Indicators**: Visible focus states
7. **Screen Reader**: Announcements for dynamic content

---

## SEO Optimization

1. **Meta Tags**: Title, description, OG tags
2. **Structured Data**: JSON-LD for rich snippets
3. **Sitemap**: XML sitemap
4. **Robots.txt**: Proper indexing rules
5. **Canonical URLs**: Prevent duplicate content
6. **Performance**: Fast load times (< 3s)
